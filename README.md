Simpe Calculator in C

This is a basic calculator program written in C that performs simple arithmetic operations such as
addition, subtraction, multiplication and division.
It is beginner friendly and demonstrates the use of conditional logic units using 'switch' statement.

FEATURES:
-Addition
-Subtraction
-Multiplication
-Division
-Error handling for division by zero
-Clean and simple code strucure

HOW IT WORKS:
1. The user enters an operator: '+', '-', '*' or '/ '
2. The user enters two numbers
3. The program performs the calculation using 'switch' case
4. The result is displayed with two decimal places

CODE FILE:

The complete code is available in
**Calculator. C** inside the repository.

HOW TO RUN:

1.Install a C compiler

2. Compile the file
   
3. Run the program

SKILLS USED:
- C Programming basics
- Input/Output handling
- switch-case statements
- Arithmetic operations
